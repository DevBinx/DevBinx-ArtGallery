# 🎨 ArtGallery
다양한 예술 작품들을 갤러리 형태로 전시한 과제제출용 웹 사이트 포트폴리오 입니다.

> 빈센트 반 고흐, 레오나르도 다빈치, 앤디 워홀 등 다양한 예술가의 작품들을
갤러리 형태로 한 눈에 볼 수 있으면 예술 작품에 대한 접근성을 개선할 수 있지 않을까?

이러한 생각이 프로젝트 개발의 기초가 되었습니다.
